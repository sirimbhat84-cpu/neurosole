NeuroSole

Smart Safety-Controlled Therapy System for Diabetic Neuropathy

---

🧠 Overview

NeuroSole is a safety-first biomedical engineering project designed to assist individuals with Diabetic Peripheral Neuropathy (DPN) using controlled pneumatic stimulation.

Unlike conventional therapy devices, NeuroSole introduces an intelligent pre-use screening system that evaluates patient conditions before allowing therapy, ensuring safe and personalized usage.

---

⚠️ Problem Statement

Existing pneumatic compression and stimulation devices focus only on therapy delivery and do not evaluate whether a patient is medically safe to use them.

This can lead to serious risks in individuals with:

- Varicose veins
- Deep vein thrombosis (DVT)
- Peripheral arterial disease (PAD)
- Cardiac conditions
- Severe neuropathy

---

💡 Proposed Solution

NeuroSole integrates a software-based medical screening interface with a hardware-controlled pneumatic system to ensure:

- Safe usage decisions before therapy
- Adaptive therapy modes (blocked / restricted / normal)
- Prevention of misuse in high-risk patients

---

⚙️ System Workflow

User → Screening Interface → Risk Analysis → Decision Engine

- ❌ High Risk → Device usage blocked
- ⚠ Moderate Risk → Low-pressure mode enabled
- ✔ Safe → Full therapy allowed

→ Pneumatic system executes controlled inflation–deflation cycles

---

🔧 Key Features

- 🛑 Pre-use Medical Screening System
- ⚠ Dynamic Risk-Based Therapy Control
- 💨 Inflation–Deflation (IPC) Mechanism
- 📊 User-Friendly Interface for Decision Feedback
- 🔒 Safety-first design to prevent harmful usage

❤️ Inspiration

This project is inspired by real-world challenges faced by diabetic patients, particularly those with multiple co-existing conditions such as neuropathy and vascular disorders.

The goal is to create a system that prioritizes patient safety before therapy, rather than applying a one-size-fits-all approach.

---

🏆 Innovation

The novelty of NeuroSole lies in its decision-first architecture:

«The device evaluates whether therapy should be allowed before it begins, reducing the risk of complications in vulnerable patients.»
